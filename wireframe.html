01 · Grid
3-Column Dashboard
grid-template-columns: 300px 1fr 220px — fixed sidebar, flexible center, fixed stats column. Profile panel spans 2 rows with grid-row: 1/3.
02 · Clock
Live JavaScript Clock
setInterval(tick, 1000) calls the tick function every 1000ms (1 second). Each call reads new Date() and updates the DOM text with the current time.
03 · Terminal
Typewriter Effect
Lines array + recursive setTimeout. Each call appends the next line to the terminal div with a delay of 600ms for commands and 80ms for output lines.
